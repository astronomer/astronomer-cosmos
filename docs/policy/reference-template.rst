Reference Template
=========================

Introduce the contents of the page. This might include:

- What feature or functionality does this reference relate to?
- Are there any existing guides demonstrating how to use it?
- Is there code source or a demo project related to it?

Reference section or entries
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Reference sections commonly need to answer the following questions. You will need to include:

- The parameter, configuration, element, function, name
- A description of what it is or does
- The default values
- (Optional) Related configurations, parameters, elements, or functionality that are affected by changing this.
